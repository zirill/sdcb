# sdcb
Shell script for certbot-docker 

* add domain
* delete domain 
* renew 
* show list all domain 
* add cron 
* romve cron


### Download
```
curl https://raw.githubusercontent.com/zirill/sdcb/main/cb.sh -O 
```

### Set attribute
```
chmod +x cb.sh
```

### add new domain
```
./cb.sh a mydomain.com
```

### delete domain
```
./cb.sh d mydomain.com
```


### help
```
./cb.sh h
```
