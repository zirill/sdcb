# sdcb
Shell script for certbot-docker 


### Download
```
curl https://raw.githubusercontent.com/zirill/sdcb/main/cb.sh -O 
```

### Set attribute
```
chmode +x cb.sh
```

### add new domain
```
./cb.sh a mydomain.com
```

### delete domain
```
./cb.sh d mydomain.com
```


### help
```
./cb.sh h
```
